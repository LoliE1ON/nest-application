## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript.
JWT strategy, Mongodb, Swagger

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
